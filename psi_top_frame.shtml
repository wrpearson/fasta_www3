PSI-Search at the U. of Virginia

UVa FASTA Server
The PSI-Search server (1) submits sequences to the NCBI PSI-Blast server, (2) retrieves the results, (3) calculates the statistical significance of the alignments using SSEARCH and PRSS, and then (4) adds the SSEARCH and PRSS statistical estimates to the PSI-BLAST output. The SSEARCH/PRSS E()-values are then used to select sequences for the next iteration.

PSI-Search seeks to emulate PSI-Blast, but currently it does not work with subsets of sequences properly.
